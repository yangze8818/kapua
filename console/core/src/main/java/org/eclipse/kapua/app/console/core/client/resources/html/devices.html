Do you want to build a programmable IoT gateway using your RaspberryPi, BeagleBone or similar open hardware?
Download and install Eclipse Kura which a offers Java-based application development framework for IoT gateways.
You will have your device connected and managed by Kapua in few minutes.

Contacting Kapua if you would like to upgrade to industrial or rugged IoT gateways running the commercial supported version of Kura, ESF - Everyware Software Framework.
